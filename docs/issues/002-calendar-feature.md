# カレンダー機能の追加 - 習慣の継続状況を可視化

## 概要
習慣がどれだけ続いているかを視覚的に確認できるカレンダー機能を追加する。

## 背景・目的
現状では習慣の継続状況を一目で把握することが難しい。カレンダー形式で習慣の達成状況を表示することで、ユーザーのモチベーション維持と継続状況の把握を容易にする。

## 機能要件

### 基本機能
- [x] 月単位のカレンダー表示
- [x] 各日付に習慣の達成状況を表示（達成/未達成/一部達成）
- [x] 連続達成日数（ストリーク）の表示
- [x] 月間の達成率の表示

### カレンダーUI
- [x] 月の切り替え機能（前月/次月）
- [x] 達成状況に応じた色分け表示
  - 全達成: 緑
  - 一部達成: 黄
  - 未達成: 灰/赤
- [x] 特定の日付をクリックすると、その日の習慣詳細を表示

### データ連携
- [x] 既存の習慣達成履歴データとの連携
- [x] 習慣ごとのフィルタリング機能

## 実装内容

### 追加ファイル
- `frontend/src/components/HabitCalendar.tsx` - 習慣カレンダーコンポーネント

### 主な機能
1. **カレンダー表示**: react-day-pickerを使用した月単位のカレンダー
2. **達成状況の可視化**: 
   - 緑: 全習慣達成
   - 黄: 一部達成
   - 灰: 未達成
3. **統計情報**: 
   - 現在のストリーク（連続達成日数）
   - 最長ストリーク
   - 今月の達成率
4. **習慣フィルタリング**: 特定の習慣のみの表示に対応
5. **日付詳細**: 日付クリックで達成した習慣の一覧を表示
